# Voice

## 基调
- 中性、可信、直接
- 不煽情,不装酷
- 第二人称("你")而非"用户"

## 原则
- 按钮文案是**动作** + **对象**:"保存设置""立即下单",而不是"确定""好的"
- 错误文案说清**原因** + **下一步**:"邮箱格式不对 — 加 @ 和域名再试"
- 空态文案是**邀请**,不是报备:"开始第一次记录 →" 而非 "暂无数据"
- 成功文案不必过度庆祝:"已保存" 足够,不要 "太棒了!已成功保存 🎉"

## Don't
- 不用"您"(除非 DS 明确指明)
- 不用感叹号(除非严肃警告)
- 不用 emoji 做功能装饰(见 craft/anti-ai-slop)
